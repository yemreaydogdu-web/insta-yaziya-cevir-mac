ZELKA SCRIBE v2.0.1

Bu repo, çalışan Insta Yazıya Çevir motorunun Zelka Scribe ürün kimliğine taşınmış halidir.

ÖNEMLİ:
- Transkripsiyon motoru, model seçimi, link indirme ve çıktı alma mantığı korunmuştur.
- Yeni arayüz, logo, app icon ve macOS build isimleri güncellenmiştir.
- Normal çıktı olarak yalnızca iki dosya üretilir:
  1) *.zamanli.txt
  2) *.duz_metin.txt

MAC BUILD:
- GitHub Actions otomatik çalışır.
- Elle build için: bash build_macos.sh

ÇIKTILAR:
- dist/Zelka Scribe.app
- dist/ZelkaScribe.dmg

Instagram linklerinde hata olursa Chrome/Safari çerezi seçilmeli ve tarayıcıda ilgili hesap açık olmalıdır.


WINDOWS BUILD
-------------
GitHub'da Actions sekmesinden "Build Windows app" workflow'unu çalıştır.
Çıktı artifact adı: ZelkaScribe-Windows-app

Yerel Windows build için:
  build_windows.bat
veya PowerShell ile:
  .\build_windows.ps1

Çıktı klasörü:
  dist\Zelka Scribe\Zelka Scribe.exe
