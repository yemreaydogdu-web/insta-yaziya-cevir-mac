MAC KULLANIYORSAN HEDEF DOSYA .EXE DEĞİL, .APP / .DMG'DİR.

Bu zip kaynak projedir. Final uygulamayı üretmenin en kolay yolu:

1) GitHub reposuna bu klasördeki dosyaları yükle.
2) src/InstaYaziyaCevir.py dosyasının güncellendiğinden emin ol.
3) Actions sekmesine gir.
4) Build macOS app > Run workflow de.
5) Artifacts bölümünden InstaYaziyaCevir-macOS-dmg dosyasını indir.
6) DMG'yi aç, uygulamayı Applications klasörüne sürükle.
